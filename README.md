# -Smart-Personal-Finance-Expense-Tracking-Mobile-App-
1. Project Overview

Project Title: Smart Personal Finance & Expense Tracker
Type: Mobile FinTech Application
Tool Used: Figma
This project focuses on designing a modern mobile application that helps users track daily expenses, monitor spending habits, and manage their personal finances efficiently.The goal of the application is to provide a simple and intuitive interface where users can easily record expenses, analyze financial patterns, and stay within their budgets.
                Designed Screens
The following mobile screens were designed for the application:
                             Login Screen
                            Dashboard Screen
                           Add Expense Screen
                          Expense History Screen
                          Budget Overview Screen
                         Profile / Settings Screen
                         Category Selection Screen
Each screen was designed with a clean layout, clear typography, and consistent spacing to create a professional user experience.
                                    Design Elements
                                       Typography
                              Primary Font: Inter / Poppins
                                    Heading Size: 24 px
                                    Body Text: 14–16 px
                                        Color System
                                   Primary Color – Blue
                                  Background – Light Grey
                                       Income – Green
                                       Expense – Red

Icons
Icons are used to represent categories like food, groceries, shopping, and transport to improve usability.
